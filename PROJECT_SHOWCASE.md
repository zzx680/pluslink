## 🎨 设计系统

### 配色方案
```
主色：#000000 (纯黑)
背景：#FFFFFF (纯白)
灰度：
  - 50: #F9FAFB
  - 100: #F3F4F6
  - 200: #E5E7EB
  - 400: #9CA3AF
  - 500: #6B7280
  - 600: #4B5563
强调色：
  - 绿色: #10B981 (成功/可用)
  - 红色: #EF4444 (错误/删除)
```

### 字体系统
```
标题：
  - 7xl (72px) - 主页大标题
  - 4xl (36px) - 页面标题
  - 3xl (30px) - 弹窗标题
  - 2xl (24px) - 卡片标题
  - xl (20px) - 子标题

正文：
  - lg (18px) - 按钮文字
  - base (16px) - 正文
  - sm (14px) - 辅助信息
  - xs (12px) - 标签
```

### 间距规范
```
组件内边距：
  - 小: p-4 (16px)
  - 中: p-6 (24px)
  - 大: p-8 (32px)
  - 特大: p-10 (40px)

组件间距：
  - 小: gap-3 (12px)
  - 中: gap-4 (16px)
  - 大: gap-6 (24px)
  - 特大: gap-8 (32px)
```

### 圆角规范
```
小元素: rounded (4px)
中等: rounded-lg (8px)
大元素: rounded-xl (12px)
特大: rounded-2xl (16px)
圆形: rounded-full
```

### 阴影层级
```
轻微: shadow-sm
标准: shadow-lg
强调: shadow-xl
最强: shadow-2xl
```

## ✨ 动画效果

### 1. 淡入动画 (fade-in)
- 透明度: 0 → 1
- 位移: 向上 10px → 0
- 时长: 600ms
- 应用: 页面加载、弹窗出现

### 2. 抖动动画 (shake)
- 左右摆动
- 时长: 500ms
- 应用: 错误提示

### 3. 脉冲动画 (pulse-subtle)
- 透明度: 1 → 0.8 → 1
- 时长: 2s 循环
- 应用: 状态指示点

### 4. 旋转动画 (spin)
- 360° 旋转
- 时长: 1s 循环
- 应用: 加载状态

### 5. 悬停效果
- 缩放: scale-105 (1.05倍)
- 上浮: -translate-y-1 (4px)
- 阴影增强
- 时长: 300ms

## 🎯 交互细节

### 按钮交互
1. **主按钮（黑底白字）**
   - 默认: 黑色背景
   - 悬停: 深灰背景 + 缩放 + 阴影
   - 点击: 缩小 (scale-95)
   - 禁用: 灰色背景 + 禁止光标

2. **次按钮（白底黑边）**
   - 默认: 白色背景 + 黑色边框
   - 悬停: 黑色背景 + 白色文字
   - 点击: 缩小
   - 过渡: 300ms

### 输入框交互
1. **默认状态**
   - 灰色背景 (bg-gray-50)
   - 灰色边框 (border-gray-200)

2. **焦点状态**
   - 白色背景
   - 黑色边框
   - 平滑过渡 300ms

3. **错误状态**
   - 红色边框
   - 红色背景提示
   - 抖动动画

### 卡片交互
1. **默认状态**
   - 白色背景
   - 浅灰边框
   - 轻微阴影

2. **悬停状态**
   - 黑色边框
   - 强阴影
   - 上浮 4px
   - 过渡 300ms
