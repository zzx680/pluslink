```bash
npm run dev

open http://localhost:3000/admin
```

### 查看 Cookie

### 查看邀请码数据
```bash

cat data/invite-codes.json | jq
```

### 清除 Cookie
```javascript
// 在浏览器控制台执行
document.cookie = 'admin_session=; expires=Thu, 01 Jan 1970 00:00:00 UTC; path=/;'
```
