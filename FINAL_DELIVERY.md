## 🚀 部署指南

### 前置条件
- Node.js >= 18
- Git 已配置
- GitHub 账号
- Vercel 账号

### 快速部署（5分钟）

#### 步骤 1: 推送到 GitHub
```bash
# 在你的终端运行
cd /Users/charlie/pluslink
git push origin main
```

如果遇到权限问题：
- 检查 SSH 密钥配置
- 或联系仓库管理员
- 或使用 HTTPS URL

#### 步骤 2: 部署到 Vercel
1. 访问 https://vercel.com/new
2. 选择 "Import Git Repository"
3. 选择 pluslink 仓库
4. 点击 "Deploy"
5. 等待 2-3 分钟

#### 步骤 3: 验证部署
访问 Vercel 提供的 URL，测试：
- [ ] 主页正常加载
- [ ] 企业端表单包含新字段（地点、工作方式、职位类型）
- [ ] 实习生端表单包含新字段（入职时间、地点、工作方式、职位类型）
- [ ] 实习生列表简化显示（仅姓名+岗位）
- [ ] 点击卡片可查看完整信息模态框
- [ ] 职位列表显示新字段标签
- [ ] 管理后台功能正常
