#### 自定义动画
```css

shake: 抖动效果（500ms）
pulse-subtle: 微妙脉冲（2s 循环）
spin: 旋转加载（1s 循环）
```

#### 交互动画
- 按钮悬停缩放（scale-105）

## 🚀 部署指南

### 快速部署（5分钟）

1. **推送到 GitHub**
```bash
git remote add origin https://github.com/YOUR_USERNAME/pluslink.git
git push -u origin main
```

2. **部署到 Vercel**
- 访问 [vercel.com/new](https://vercel.com/new)
- 导入 GitHub 仓库
- 点击 Deploy

3. **完成！**
- 获得部署 URL
- 访问 `/admin` 生成邀请码
- 开始使用
